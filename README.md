# jammer
A extremely compact signal jammer made using a esp 32! 
# features
This project allows an ESP32 board to control four nRF24L01+ modules for RF jamming (Bluetooth jammer, BLE jammer, WiFi jammer, RC jammer, ...) or channel sweeping across multiple 2.4GHz bands
